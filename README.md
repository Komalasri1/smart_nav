Smart Navigation System for EV Charging

Smart Navigation System for EV Charging is an intelligent web-based application that helps Electric Vehicle (EV) users find optimal routes to nearby charging stations based on availability, distance, user preferences, and ratings. The system enhances the EV driving experience through real-time route optimization and smart charging station recommendations.

---

Features

- ⚡ Smart EV Charging Station Navigation
- 📍 Real-Time Route Optimization
- 🗺️ Interactive Map Integration
- 🔋 Nearby Charging Station Detection
- 📊 Distance Calculation using Haversine Algorithm
- ⭐ Charging Station Ratings & Preferences
- 📌 Reverse Geocoding for Accurate Locations
- 📅 Charging Slot Availability Handling
- 🌱 Sustainable Driving Support
